⚽ Transfermarkt Player Performance Dashboard

An advanced football analytics platform that scrapes Transfermarkt data, performs machine learning clustering analysis, and generates interactive visualizations to discover player archetypes and performance patterns.

🎯 Data Collection

Automated web scraping from Transfermarkt covering multiple leagues including Premier League, La Liga, Bundesliga, Serie A, and Ligue 1. Extracts player market values, positions, ages, and club affiliations.

🧠 Machine Learning

KMeans clustering algorithm groups players by style proxies derived from position, age, market value, and league strength. PCA dimensionality reduction enables 2D visualization of complex player profiles.

📊 Interactive Visualizations

Beautiful Plotly and Seaborn charts showing player clusters, position distributions, and market value analysis. Hover over data points to explore individual player details dynamically.

Dashboard Output

The dashboard.py script generates comprehensive analytics including player clustering, CSV datasets, PNG visualizations, and an interactive HTML dashboard.
